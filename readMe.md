表格
<br id="bg">
表格：用|分隔单元格，表头下需要写|-|，内容写|-|下用||包括  
:-左对齐的内容和表头,:-:居中的内容和表头,-:右对齐的内容和表头
|表头1|表头2|表头3|
|:-|:-:|-:|
|内容1|内容2|内容3|
|:-左对齐的内容和表头|:-:居中的内容和表头|-:右对齐的内容和表头|
